# 바카라 게임
## 네트워크 소켓, 멀티스레드 ,mutex 개념 실습을 통한 심화 학습
## C언어 / Linux(Ubuntu)
### 현재 구현한 기능  
- 뱅커 / 플레이어 배팅
- 클라이언트는 시작 금액 설정
- 서버는 참가한 클라이언트가 모두 배팅해야 게임 시작
- 일정금액이하(돈을 다 잃으면) 강퇴 기능
- 서버는 수익을 알 수 있다.

### 앞으로 추가 해볼 기능
- 클라이언트들 회원가입/로그인 구현 ( 나중에 토이프로젝트로 react로 구현 )
- 서버는 file로 저장 후 money도 불러오는 기능 구현
- 사이드 배팅인 tie/pair 구현
